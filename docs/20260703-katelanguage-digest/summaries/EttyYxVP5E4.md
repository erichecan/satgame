# AP English Literature / AP 英语文学（人物刻画与 AP Lit 写作）
- 频道：Kate Language ｜ 时长：21:08 ｜ [原视频](https://www.youtube.com/watch?v=EttyYxVP5E4)

## 📊 执行摘要
这是 AP 英语文学与写作课，系统讲**人物刻画（characterization）**及 **AP Lit 论文写作**。先梳理 AP Lit 关于人物的标准与术语，引入"人物冰山（character iceberg）"模型，用《了不起的盖茨比》分析 Tom Buchanan，再给出 AP Lit 三段式论文模板（author presents… in order to… ultimately illustrating theme），并介绍三道 essay 题（诗/散文/自选书）与限时写作策略。

## 📝 详细总结（带时间戳）
- **[0:00] AP Lit 人物三标准**：①描写/对话/行为揭示人物；②人物通过用词、细节、思维组织、决定与行动流露**视角与偏见（perspective/bias）**；③可从人物的行动或不作为**推断动机（motivation）**。
- **[1:14] 关键术语**：cultural norms（文化规范）、conflict（冲突，分**外部**：人vs外界；**内部**：内心两种冲动，冲突驱动故事）、protagonist（主角，中性词，未必是好人）、antagonist（对立力量，未必是人/坏人）、**foil（反衬角色）**。
- **[2:47] 刻画途径**：描写（外貌/穿着/言谈/走路/思想）、对话与内心独白（体现文化/社会经济地位/价值观/智力）、行为（"行动胜于言语"）。
- **[4:17] 人物冰山（character iceberg）**：水面上=行动/言语/思想；水面下=动机/恐惧/家庭/关系/社会地位等**驱动因素**。要引导学生**从表层深入底层**理解复杂人物。
- **[5:44] 分析引导问题**：人物如何被描写？名字是否有深意？背景/动机/欲望如何影响言行？冲突揭示了什么？反衬角色凸显了哪些价值？言行与内心是否矛盾？
- **[7:39] 实操：分析《盖茨比》Tom Buchanan**：用"文本证据 + 形容词"填表——sturdy（intimidating 令人生畏）、supercilious/arrogant eyes（prideful 傲慢）、gruff husky voice/fractiousness（harsh 粗粝）、cruel body、riding clothes、eyes flashing restlessly（nervous/impatient）、用手臂拨动 Nick（masculine/overpowering 强势）。
- **[15:16] AP Lit 论文模板**：**"In [作品], [作者] presents… in order to… ultimately illustrating [主题]"**——把 prompt 三部分与主题串起来。给出诗（The Century Quilt）、散文（Cherry Bomb、Tell-Tale Heart）、自选书（Of Mice and Men）多个范例。
- **[15:36] 三道 essay 题**：Q1 诗歌、Q2 散文（本课重点）、Q3 开放式（自选读过的书答题）。
- **[19:12] 限时写作策略（每题约 40 分钟）**：**直接把 thesis 当引言**（省时）；每个主体段=主题句（回扣 thesis）+ 具体引文证据 + 分析评论（解释该例为何重要、作者如何借它构建意义）+ 收尾，"证据+分析"至少重复一轮。
- **[20:45] 写作步骤**：先用人物冰山**搜集证据**，再成文。

## 💡 主要洞察
- 人物刻画要用"冰山模型"：从表层言行深入底层动机/恐惧。
- AP Lit 论文靠模板成文："作者 presents… in order to… ultimately illustrating 主题"。
- 限时写作省时法：thesis 即引言，主体段走"主张—证据—分析"循环。

## 📚 概念与术语
- **Characterization / character iceberg（人物刻画/人物冰山）**。
- **protagonist / antagonist / foil（主角/对立方/反衬）**、**conflict（内外冲突）**。
- **AP Lit 论文模板**：presents… in order to… ultimately illustrating theme。

## ✅ 值得记住
- 分析人物先填"文本证据+形容词"表，再由表及里想动机。
- AP Lit 作文套模板、thesis 当引言、主体段走"主张—证据—分析"。
