# SAT 强化班 SAT Advanced（Digital SAT 图表题）
- 频道：Kate Language ｜ 时长：18:10 ｜ [原视频](https://www.youtube.com/watch?v=uPP3aZDqLqc)

## 📊 执行摘要
这是 SAT 强化班，专讲 **Digital SAT 的图表题（graphic/informational graphics）**。老师梳理题目格式（图 + 迷你短文 + 最后一句问题/填空）、常见话题（约 90% 来自科学）、两种题型（补全句子 / 概括趋势），并给出四步解题法（先读题→读文找 claim/hypothesis/conclusion→分析图表 title/labels/patterns→从 A 到 D 排除）。用"不同职业评价差异（varied widely）"等真题演示，并对比新旧题型难度。

## 📝 详细总结（带时间戳）
- **[0:39] 题目格式（三件套）**：先给一张图（柱状图/折线图/饼图/表格），再给一段**很短的迷你短文（通常一段）**，最后一句常是**问题或待填空白**——三部分同时出现。
- **[1:54] 话题**：约 **90% 图表题来自科学类**（自然科学：生物/植物/化学/物理/天文；社科：商业/经济/心理/人类学）；**诗歌、小说后不会有图表题**。
- **[2:52] 两种题型**：①**补全句子**（补上段落最后一句的空白，如"In conclusion, ___"）；②**概括信息**（综合图表与短文，二者**同等重要、缺一不可**）。
- **[4:25] 四步解题法（关键）**：
  1. **先读题**——别一头扎进图表。每题仅约 **70 秒**（英文），明确"要找什么"。（数学题时间更充裕，约 90 秒+，图表题不必担心时间。）
  2. **读短文**——找 **claim（主张）/ hypothesis（假设）/ conclusion（结论）/ findings（发现）**，**不是**实验步骤。
  3. **分析图表**——花几秒看懂 **title、labels、keys** 各代表什么，**找 pattern（趋势：上升/下降、柱状对比）**，尤其概括题。
  4. **看选项，从 A 到 D 逐一排除**——出题人有时把正确答案放 A 帮你省时；**务必从上到下**，别从 D 倒着看浪费时间。
  - 时间分配：词汇、语法（标点约 10–20 秒）要**快**，把时间省给难度更高、分值更高的图表题。
- **[9:10] 真题①（职业评价）**：题干"participants' evaluation **varied widely** depending on occupation, for example ___"。**关键词**：which occupation、varied widely。故答案须体现**不同职业间的大差异**——
  - A 正确：导游 82% vs 外科医生 16%，职业不同且差异巨大。
  - B/D 错：只有**一个职业**，看不出"varied widely"。
  - C 错：虽两个职业，但两者比例**相同（都 9% 中立）**，**无差异**——不仅要"不同职业"，更要"大差异"。
- **[14:39] 真题②（阿尔及利亚数据）**：定位关键词→回图找对应三角/数据点（42→72）→属"旧 SAT 风格"的**找具体数字**题，较易。
- **[15:36] 新旧题型对比**：找具体数字是新 SAT 的**基础技能**，新题在此之上更进一步（选项只给不同数字/更刁钻）；旧题句子结构更复杂反而更难。**概括趋势题**（如"从 1000 降到 500 属下降趋势"）要选**覆盖完整年份区间**的最佳概括（A 覆盖 2010–2019 优于只覆盖 2010–13 的 D）。

## 💡 主要洞察
- 图表题=图+短文+问题三件套，短文与图表同等重要，缺一不可。
- 四步法：先读题→读文找 claim/结论→analyze 图表找 pattern→A 到 D 排除。
- 抓题干关键词（如 varied widely）决定答案方向——不仅要"不同"，更要"大差异"。
- 时间管理：语法词汇求快，把 70 秒省给高分值的图表题。

## 📚 概念与术语
- **Graphic questions（图表题）**：Digital SAT 题型，科学话题为主。
- **claim / hypothesis / conclusion / findings**：读短文时要抓的要点。
- **Pattern（趋势）**：上升/下降/对比，概括题的核心。
- **排除法（A→D）**：从上到下逐项排除。

## ✅ 值得记住
- 图表题永远先读题、再读短文、再看图、最后 A 到 D 排除。
- 抓题干关键词定方向（varied widely=要找大差异）。
- 概括趋势题选覆盖完整区间的最佳概括。
