## 📊 执行摘要
这是"全能阅写 Level 1"低龄段英语课的录播片段，展示了一堂融合"写作 prompt + 读文找信息 + 语法点"的互动课。老师以"描写你理想中的卧室"作写作引子，再带学生读多伦多与蒙特利尔的对比短文、练习提取信息（共同点/差异、人口构成），最后集中操练英语缩写（contractions）的语法规则。整体体现该级别"读—写—语法"三合一、以口头互动为主的教学风格。

## 📝 详细总结（带时间戳）
- **[0:00] 写作引子**：prompt 是"描写你想要的卧室"——引导学生思考想改变什么（墙色、床的大小、增加物品等），激活表达欲。
- **[0:36] 读文找信息**：读关于多伦多和蒙特利尔的两段文字，找"共同点与差异"。选这两座城是因为都是加拿大知名大城。任务不是写整篇 essay，而是先学会**定位信息**。
- **[1:29] 具体信息点**：蒙特利尔约 200 万人口、主要语言是**法语**（可答 "French" 或 "mostly French"，因原文给了较高的法语使用比例）。
- **[2:06] demographic（人口构成）概念**：指"什么样的人住在这座城"。以多伦多为例——50% 居民属于"可见少数族裔（visible minority）"，即非白人（含亚裔、拉丁裔、黑人等各种有色人种）。
- **[3:04] 语法点：缩写 contractions**：定义=两个词合并变短。带学生识别与改写——we are not→we weren't、is not→isn't、have not→haven't、was not→wasn't、will not→won't。反复强调**撇号（apostrophe）替换被省略字母的位置**。

## 💡 主要洞察
- 低龄英语课把"读、写、语法"揉进同一堂课，用贴近生活的话题（卧室、城市）降低门槛。
- 阅读训练从"先会定位信息"起步，而非一上来就写整篇作文。
- 语法（缩写）通过大量即时改写句子来内化，而非讲规则。

## 📚 概念与术语
- **Demographic（人口构成）**：一座城市/地区住着什么样的人。
- **Visible minority（可见少数族裔）**：非白人群体。
- **Contraction（缩写）**：两词合并、用撇号替代省略字母（won't、isn't 等）。

## ✅ 值得记住
- 教缩写时，重点让学生看清撇号"替换掉哪个字母、放在哪个位置"。
- 阅读任务先练"定位/提取信息"，再谈成文。
