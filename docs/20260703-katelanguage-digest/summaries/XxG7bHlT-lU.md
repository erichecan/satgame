# IB 体系课程 TOK 先导班 试听（Theory of Knowledge）
- 频道：Kate Language ｜ 时长：10:18 ｜ [原视频](https://www.youtube.com/watch?v=XxG7bHlT-lU)

## 📊 执行摘要
这是 IB TOK（知识论）先导班试听片段，介绍该课与普通读写课的三点不同：①做**真正的学术研究**（讲深度、学术诚信、autonomy 自主）；②读**大量不同体裁**的学术文章（带特定阅读焦点）；③写作是**自主命题的独立研究论文**，而非给定 prompt 的作文。目的是让学生学会"带着特定焦点阅读、并从零到尾主导自己的研究"。

## 📝 详细总结（带时间戳）
- **[0:37] 特点一：真学术研究**。常见误区是"研究=上网 google 一下"；学术研究**不能肤浅、经不起推敲**。要点：
  - **深度（depth）**：研究要透彻。
  - **学术诚信（academic integrity）**：理解它到底涵盖什么、如何在研究中落实。
  - **自主（autonomy）**：auto=自我驱动——**你自己决定研究什么、聚焦什么、如何推进**，你是自己研究的"节拍器（pacemaker）"。这正是 TOK 区别于普通读写课之处。
- **[3:11] 特点二：体裁多样（variety of passages）**。会读很多"technical（在某领域深入探究、你可能从没听过）"的学术文章——如 NASA 某新天文技术项目。即便题材陌生、术语繁多，也要学会**带着特定焦点抓住重点**，而非逐字读懂每个词。
  - **[5:21] 选材范围**：人性（社科）、自然科学、科学实验/理论、政治、社会经济地位、公平/自由/社会议题、教育、医疗，以及当下热点（如疫情等争议话题）。选材**可调整、可定制**，随时可引入新报道（如《纽约时报》评论）或学术期刊文章。
- **[7:13] 特点三：写作=独立研究论文**。关键区分：
  - **带 prompt 的作文（essay with prompt）**：题目/方向已给（如议论文），你顺着线索深入。
  - **独立研究论文（independent research-based paper）**：**从选题到成文全由你自己负责**——难点在于如何找到切入角度、挖得够深、给出有力论证。这呼应"什么构成强论证""学术写作 vs 创意/个性化写作的区别"。
- **[9:57] 教学节奏**：每节课引入一个新话题或补充材料，帮助逐步掌握"带特定焦点阅读"。

## 💡 主要洞察
- TOK 的核心是 **autonomy**：学生自主命题、主导研究全过程。
- 读陌生学术文章的关键不是全懂，而是"带着焦点抓重点"。
- 独立研究论文比给定 prompt 的作文更难，难在自选角度并挖深。

## 📚 概念与术语
- **TOK（Theory of Knowledge，知识论）**：IB 体系课程。
- **Autonomy（自主）**：自我驱动，自定研究方向。
- **Academic integrity（学术诚信）**：研究须遵守的准则。
- **Essay with prompt vs. independent research paper**：给定题目的作文 vs 自主命题的研究论文。

## ✅ 值得记住
- 学术研究要深、要守诚信、要自主。
- 读陌生题材文章时带着"特定焦点"去抓重点。
- 练习自选题目、自己找研究角度并挖深。
