## 📊 执行摘要
这是 SSAT 伴学营片段，示范一道**词汇多选题**的完整解法：题干词 intolerant，选项 dogmatic / emaciated / amenable。老师用"前缀 prefix–词根 root–后缀 suffix"三段拆解逐词分析，并强调最终要**依据词义（而非词性）**选出答案。过程中穿插大量构词法与美国宪法/政治小知识。

## 📝 详细总结（带时间戳）
- **[0:00] 通用法：拆三段**。遇到又难又长的词，永远拆成 **前缀 + 词根 + 后缀**。
- **[0:56] 拆 intolerant = in + toler + ant**：
  - 前缀 **in-** 有两层义：①"在里面"（方位）；②更重要的——**否定**。同族否定前缀还有 im-/il-/ir-（immature、immortal、illegitimate、irresponsible）——加在词前会**彻底反转词义**。
  - 词根 **toler**=忍受、容忍（tolerance）。
  - 后缀 **-ant**：多决定词性（常为名词指人，此处作形容词）。
  - 合起来 = 不愿容忍 + 主观武断 → 含"轻蔑、看不起"之意。
- **[4:02] dogmatic**：dog（"固执如狗"）+ -atic → 教条的、固执的（坚持未经证实的原则/理论），是 stubborn 的近义词。
- **[5:21] emaciated = e/ex（向外）+ maci（瘦/词根）+ -ated**：因疾病、饥饿或寒冷而**极瘦**，带消极义。
- **[6:33] amenable = amend（修改/调整）+ -able**：可被修正/可调整的。借此讲**美国宪法**：宪法是根本大法，后来的总统/政客不能否定它、只能"修正（amend）"→ amendment（修正案，如著名的 First Amendment）；并延伸讲参议院/众议院、435 名众议员按州大小分配等政治常识。
- **[9:43] 定答案**：三个选项都是"客观"倾向的词，须**据词义而非词性**判断——intolerant 最接近 **dogmatic**。

## 💡 主要洞察
- 攻长难词的万能法：拆前缀/词根/后缀，再合并猜义。
- 否定前缀 in-/im-/il-/ir- 会彻底反转词义，是高频得分点。
- 词汇题最终按**词义**定答案，别被相同后缀/词性误导（后缀相同可能是陷阱）。

## 📚 概念与术语
- **前缀 in-/im-/il-/ir-**：否定，反转词义。
- **-ant / -atic / -able 等后缀**：多决定词性。
- **intolerant / dogmatic / emaciated / amenable**：不容忍的、教条的、极瘦的、可修正的。
- **amend → amendment（宪法修正案）**：由 amenable 的词根延伸。

## ✅ 值得记住
- 见到难词先拆三段：prefix + root + suffix。
- 词汇题据词义选，不据词性选。
