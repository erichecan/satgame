# SAT 1550+ BOOTCAMP（SAT 强化冲分班）
- 频道：Kate Language ｜ 时长：19:07 ｜ [原视频](https://www.youtube.com/watch?v=cCX3pg8Muno)

## 📊 执行摘要
这是面向 1450→1550+ 学生的 SAT 冲分强化班第一课。老师定下四大目标：大量**限时真题练习**、系统化**分析题技巧**（秒辨题型、识别错误/正确选项特征）、专攻**逻辑阅读题**、训练**思维导图速读**。并强力推荐记**错题本（error log）**分析个人错误模式，随后用真题演示词汇题、main purpose 题的解法与"标准差 SD 越大越分散"的判断。

## 📝 详细总结（带时间戳）
- **[0:18] 四大目标**：①**限时练习**——每次一个 English module 限时 32 分钟，材料全用去年/今年**真题**（College Board 题库有限，未来很可能撞见相同/相似题，故务必把讲过的每题吃透）；②**分析题技巧**；③专攻**逻辑阅读题**；④教**思维导图/骨架速读**抓文章逻辑。
- **[1:54] 分析题技巧细分**：**3 秒识别题型**（主旨/细节/推断/支持题）→ 对应系统解法；掌握 **College Board 错误选项的共性特征**，以及正确答案的特征（正确答案常是**换词改写/paraphrase**，绝不用原文原词）。
- **[4:07] 错题本（error log）**：把所有错题记下来分析。虽耗时但极有用——
  - **发现个人错误模式**：例如某学生总倾向选"more valid/更有道理"的选项，这类错误叫 **fictitious comparison（虚假比较）**——原文根本没比较 A 与 B，但选项造出一个"看似学术"的比较来设套。
  - **追踪进步、增强信心**：同类错误随练习减少（如数学从几何/三角常错到渐少）。
- **[7:21] 词汇题演示（accentuate）**：答案 B accentuate=强调/突出（highlight）。College Board 爱用 **contrast（对比）**结构——前文"排斥浪漫主义风格"，此处"突出主体的美与英雄气"，方向相反→选强调义；干扰项 obscure（使不明显）为反义。
- **[8:38] main purpose 题（注意是 purpose 不是 idea）**：选项都是"动词+宾语"的动作短语。解法：**重点读"动词 + 其后名词宾语"**；通读全文但聚焦**首句（主题句）与末句**。
  - **but/however 后半为重**：见"逗号 + but/however"聚焦后半；见 although 从句聚焦逗号后主句。
  - **双破折号（—…—）= 双逗号**：中间部分不重要、可略。
  - 找到句子**主干**后**定基调（tone）**（本例为负面：某做法"insufficient/不足以"）；正确答案 D "discuss the difficulty"匹配"负面"。正确答案**不用原文同词/同形容词**，而用**基调一致的同义词**。排除 B（challenging 虽负面但涉及"比较"，而原文无比较）、A/C（中性）。
- **[13:40] 标准差（SD）速判**：SD 衡量数据相对均值的离散程度。**SD 越大越分散、越小越集中**（55555→SD≈0；5 10 15 20 25→SD 大）。SAT 只让你**比较不同数据集**的 SD 大小、从不让你算具体值——记住"大=分散、小=集中"两句即可，用点图判断谁更向两端分散。
- **[18:27] 数学做题建议**：**先读懂题再算**、单词题**读两遍**，否则算半天答案还错。

## 💡 主要洞察
- 冲高分的关键是**错题本 + 分析个人错误模式**（如"虚假比较"偏好）。
- 正确答案=同义改写、基调一致；错误答案有 College Board 的固定套路（虚假比较、原词陷阱、离题）。
- main purpose 题读"动词+宾语"、抓首末句、but/however 后半、双破折号中间可略、先定基调。
- SD 只需记"大=分散、小=集中"，用于比较而非计算。

## 📚 概念与术语
- **Error log（错题本）**：记录并分析错误模式。
- **Fictitious comparison（虚假比较）**：选项造出原文没有的比较。
- **main purpose 题**：读"动词+宾语"、首末句、定基调。
- **Standard deviation（标准差）**：大=分散、小=集中。

## ✅ 值得记住
- 冲分必备错题本，找出自己的错误模式。
- 正确答案找"换词不换意 + 基调一致"；警惕虚假比较与原词陷阱。
- SD 题记两句话：大=分散、小=集中。
- 数学先读懂题再算，单词题读两遍。
