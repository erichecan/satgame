# 阅写 Lv4 / 文法 Lv3（议论文·利益相关者分析）
- 频道：Kate Language ｜ 时长：19:18 ｜ [原视频](https://www.youtube.com/watch?v=hinqUcm1FiY)

## 📊 执行摘要
这是"全能阅写 Lv4 / 文法 Lv3"课，教议论文用**利益相关者（stakeholder）头脑风暴**搭建主体段。以"胖乘客是否该买两张机票/影票"为例，列出所有 stakeholder 及其立场与理由，制成表格来生成 body paragraph。中途借"discrimination（区分 vs 种族歧视）"一词做批判性思维训练，揭示词义被"洗脑式"窄化的现象。

## 📝 详细总结（带时间戳）
- **[0:00] shareholder vs stakeholder**：**shareholder（股东）**对实体有实际所有权；**stakeholder（利益相关者）**范围更大，含股东，也含"无所有权但有利害关系"者（如员工）。写议论文要想"所有利益相关方"。
- **[1:16] 案例题**："larger passengers 是否该买两张机票/影票？"先头脑风暴谁是 stakeholder：胖乘客、普通乘客、航空公司、飞机制造商（Boeing/Airbus，与航司不同）、影院（发行商/影院业主）、新闻媒体等。
  - **[3:00] 追问边界**：空乘算不算 stakeholder？分析后认为关系弱（新规须在购票时执行，乘客登机前已买票，空乘无从介入）。
- **[4:16] 制表（关键工具）**：为每个 stakeholder 列 **立场（position）+ 理由（reason）**：
  - 胖乘客：**反对**——不公平、有歧视、多花钱。
  - 普通乘客：偏支持/中立——一个被忽略的点是"会让瘦乘客对自己感觉良好"。
- **[4:16] discrimination 词义辨析（批判性思维）**：老师问"歧视是坏事吗？"多数学生条件反射答"是"。实则 **discrimination 本义=区分/分辨（distinguish）**，与 incriminate 相对；人每天都在做"discrimination（在选项间取舍）"，无法避免。它被**窄化等同于种族歧视**，是"编程/洗脑"的结果——深层大脑不经点拨不会启动。
- **[9:44] 航空公司立场（用证据推理）**：航司为**利润**多半**支持**。证据——近 10–15 年座位越做越小、越挤（为塞更多人）；若允许对胖乘客加价，能减少空座、增收。
  - **[13:42] 为何还没实行？**：这是一种"胁迫（coercion）"手段；商务舱常仅为经济舱两倍价甚至更低，且商务舱座位不相邻、买两张也不现实——所以真按此收费，胖乘客要么不来、要么被迫多付。
- **[15:17] 落到 body paragraph**：用"stakeholder—立场—理由"信息成段：先写航司为招揽客流已把座位缩小、增加了对胖乘客的压力……下一段再写"航空公司想要什么（此举对其利益的好处）"。
- **[19:07] 布置练习**：对下一题"网络霸凌的学生是否该停学"做同样的 stakeholder 分析。

## 💡 主要洞察
- 议论文用 **stakeholder 头脑风暴 + 立场/理由表格**系统生成主体段，覆盖各利益方。
- 批判性思维要拆穿"被窄化/被洗脑的词义"（discrimination 本义是"区分"）。
- 分析各方立场要**用证据推理**（座位变小→航司逐利），而非想当然。

## 📚 概念与术语
- **Shareholder vs stakeholder（股东 vs 利益相关者）**：后者范围更大。
- **Stakeholder 表格**：position（立场）+ reason（理由）。
- **discrimination（区分）**：本义辨别，非等于种族歧视。
- **Coercion（胁迫）**：利用对方无选择余地施压。

## ✅ 值得记住
- 写议论文先列全 stakeholder，各配"立场+理由"表，再成段。
- 警惕被窄化的词义，凡词先想本义。
- 判断各方立场用证据（如航司座位变小佐证其逐利动机）。
