# SSAT Vocabulary Course（SSAT 词汇·词根拆词法）
- 频道：Kate Language ｜ 时长：33:30 ｜ [原视频](https://www.youtube.com/watch?v=5vNYdVJFRXU)

## 📊 执行摘要
这是 SSAT 词汇课第一讲，系统建立"**词根拆词法**"：区分 parts of speech（词性）与 word parts（构词），讲透**前缀 prefix / 词根 root / 后缀 suffix** 三部分——**前缀+词根定词义、后缀定词性**。用 unchangeable、incontrovertible、metamorphosis 演示拆长词，用"翻转联想"记短词，并系统讲"表人后缀"（-an/-ist/-er）与词根 nov=new，配褒贬标记。

## 📝 详细总结（带时间戳）
- **[0:00] 两个概念**：**parts of speech（词性）**=名/动/形/副/连词；**word parts（构词）**=几乎每个英语词都能拆成 **prefix + root + suffix**。
- **[1:53] 三部分定义与作用**：
  - **前缀（prefix，词首几字母）**：带基本含义。例 re-=再/回（reread、review、revise）；un-=否定（不）；com-/con-=一起/共同（company、commitment）。（"prefix"里的 pre 本身就=之前）
  - **词根（root，中部）**：承载词的**主要含义**。
  - **后缀（suffix，词尾）**：多决定**词性**（-tive→形容词 active；-able→形容词；-osis→名词过程；-ist/-er/-an→表人名词）。
  - **核心口诀：前缀+词根告诉你"意思"，后缀告诉你"词性"**——据此可猜出没见过的长词，命中率很高。
- **[7:54] 拆词演示**：
  - **unchangeable** = un（不）+ change（变）+ able（能）→ 不能改变的（形容词）。
  - **incontrovertible** = in（不）+ contra（反对）+ vert（转）+ ible → "不能被反转/反驳"→ **确凿无疑、绝对正确**（虽带 in- 却是**褒义**）。
  - **metamorphosis** = meta（变，如公司 Meta）+ morph（形状）+ osis（过程）→ **变形的过程**（毛毛虫→蝴蝶）。
  - **结论：词越长越好拆、越好猜**；真正的噩梦是 2–3 字母的短词。
- **[14:33] 短词联想法**：**mar**（毁坏/破坏，动词）——把 m 翻转成 w→接近 war（战争=破坏/废墟），据此记忆。
- **[19:38] 本课主线：表人的后缀（people nouns）**：
  - **-an**：Republican（共和党支持者）、**charlatan（骗子/招摇撞骗者，负）**、**partisan（党徒/坚定支持者；party=政党而非派对，中性偏语境）**、**abecedarian（初学者=只懂 ABC，负/中）**——真出现在 SSAT 真题里的 novice。
  - **-ist**：scientist；**-er**：也表人。
- **[31:39] 词根 nov=new**：novice（新手/初学者）、innovation（创新）、novelty、novel（作形容词=新颖的，与"小说"义不同——一词多性多义是词汇的难点）。
- **[25:31] 褒贬标记法**：老师用 +（褒）/ −（贬）/ ／（中性）标注每个词（如 charlatan 为负）。

## 💡 主要洞察
- **前缀+词根定义、后缀定词性**——这是拆词猜义的总纲。
- **长词更好拆猜，短词才是噩梦**（用翻转/联想记短词，如 mar↔war）。
- 即使带否定前缀（in-），整体也可能是褒义（incontrovertible=确凿）——要拆完整体判断。
- 表人后缀 -an/-ist/-er 能快速判断"这是个人/职业"。

## 📚 概念与术语
- **Word parts：prefix / root / suffix**（前缀/词根/后缀）。
- **口诀**：前缀+词根=词义，后缀=词性。
- **词根/词缀**：re-（再）、un-/in-（否定）、com-/con-（共同）、contra（反对）、vert（转）、meta（变）、morph（形）、-osis（过程）、nov（新）、-an/-ist/-er（表人）。
- **charlatan（骗子）/ novice / abecedarian（初学者）**。

## ✅ 值得记住
- 见长词先拆 prefix+root+suffix，前缀词根猜义、后缀判词性。
- 短词用翻转/联想记（mar↔war=毁坏）。
- 拆完看整体判褒贬，别被 in-/un- 误导。
