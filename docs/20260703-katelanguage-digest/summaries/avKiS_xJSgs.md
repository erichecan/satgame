# Public Speaking in 10 Scenarios（公共演讲·修辞三角与说服模式）
- 频道：Kate Language ｜ 时长：17:53 ｜ [原视频](https://www.youtube.com/watch?v=avKiS_xJSgs)

## 📊 执行摘要
这是"公共演讲"课，通过朗读无意义诗《Jabberwocky》训练"不靠理解、靠自信与语气表演"的口才；讲解**修辞三角（作者—受众—主题）**与三种说服模式 **Logos/Pathos/Ethos**（用抢答游戏辨认），并让学生用 Pathos+Logos 为"四天上学周"辩论、做 Shark Tank 推销练习，强调**用词选择（word choice）**对说服的作用。

## 📝 详细总结（带时间戳）
- **[0:00] Jabberwocky 朗读练习**：这首刘易斯·卡罗尔的诗充满生造词、**本就不需要读懂**。练习目的是**表达/表演（presentation）**——用语气、情绪、肢体动作传递"斩杀怪兽"的动作与兴奋。呼应上节"说胡话也要听起来自信"。标点（感叹号）、计数（one two）都是诗人用来传达动作的线索。
- **[3:36] 修辞三角（rhetorical triangle）**：由**作者/说者（编码信息）—受众（解码信息）—主题**三要素构成，三者平衡时沟通最有效。作者须理解主题、包装信息，受众才能正确解码。
- **[4:52] 三种说服模式（辨认游戏"Name that Appeal"）**：
  - **Ethos（信誉/权威）**：诉诸说者可信度。例"兽医说德牧适合我们活跃的生活方式"——兽医是权威职业，故为 ethos；"Mr. Clean 站在洁净厨房"——虚构可信形象，也是 ethos。
  - **Logos（逻辑/事实）**：诉诸数据事实。例"十位牙医九位推荐高露洁"——用统计支撑。
  - **Pathos（情感）**：诉诸情绪。例"别被困在没有流量的套餐里，现在换购省大钱"——制造紧迫感、诉诸感受。
- **[9:22] 辩论练习**：为"四天上学周"用 **Pathos（学业压力大、易致压力性疾病/抑郁）+ Logos（一天 24 小时被上学、睡眠、吃饭挤占，休息时间不足的量化）** 说服。学生实战演练（含"校服很贵"等展开）。
- **[15:51]（延伸讨论）校服争议**：反方观点——校服限制表达自由、强推从众、并不能减少霸凌或提升出勤/成绩。
- **[15:51] Shark Tank 推销活动**：选一个物品当作自己发明的产品，向"投资人（老师）"推销。**关键是 word choice**——范例"把一粒盐说成珠宝：完美方形、晶莹、opaque、精致"，用措辞打动受众。作业面向"投资人"，要证明产品值得投资。

## 💡 主要洞察
- 演讲表演力可脱离"完全理解内容"——靠自信、语气、肢体传达。
- 说服=修辞三角平衡 + 恰当运用 Ethos/Pathos/Logos。
- **用词选择**是说服的核心武器（把盐讲成珠宝）。

## 📚 概念与术语
- **Rhetorical triangle（修辞三角）**：作者—受众—主题。
- **Ethos / Pathos / Logos**：信誉 / 情感 / 逻辑三种说服模式。
- **Word choice（用词选择）**：影响说服效果的关键。

## ✅ 值得记住
- 朗读/演讲重在自信与语气，不必先全懂内容。
- 判断说服模式：诉诸权威=ethos、诉诸情绪=pathos、诉诸数据=logos。
- 推销/说服时刻意打磨用词。
