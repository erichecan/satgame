# PRD：NYT 风格词汇游戏模块

**版本**：v1.0  
**日期**：2026-06-22  
**产品背景**：SAT Game —— 为 Isabella（Grade 9，加拿大）打造的每日游戏化 SAT 备考 APP

---

## 一、模块定位

NYT 词汇游戏模块的核心目的是：**帮助学习过程中产生的生词，同时扩充 SAT 词汇量**。

游戏的趣味性是内容的承载方式，而非目的本身——玩法本身就是练习。

---

## 二、进入流程（学词前置锁）

每次进入 NYT 词汇游戏区，必须先完成当日词汇学习，才能解锁三个游戏：

```
[进入词汇区]
     ↓
[词汇学习：50 张词义卡片翻转展示]
     ↓ 学完全部 50 词
[解锁：今日三个游戏]
     ├─ Clusters（Connections 式）
     ├─ Five（Wordle 式）
     └─ Strands（Strands 式）
```

**当日已完成学习**：下次进入直接显示游戏（不重复强制学词），直到次日刷新。

---

## 三、词汇池（游戏范围）

游戏选词来源于三类词汇的并集：

| 来源 | 说明 |
|------|------|
| 当日 50 个 SAT 词 | 每日 DailyWordSet 中配置的 50 词 |
| 关联词汇 | 这 50 词的同义词、反义词、SAT 考试风格的关联词 |
| 生词本 | 用户 VocabNote 中到期需复习的词 |

关联词汇的选取标准：
- 符合 SAT 阅读写作考试词汇范围
- 与核心 50 词有语义关联（近义/反义/同主题）
- 本身也是 SAT 高频或中频词

---

## 四、词汇学习页面（/vocab/study）

### 4.1 设计原则

- **Generation > Recognition**：尽量让用户主动回忆，而非被动识别
- **英英优先**：默认展示英文释义，底部提供「译成中文」按钮按需使用
- **不强制记忆测试**：学习阶段只要求看过，测试留给游戏完成

### 4.2 词卡格式

每张词卡包含：

```
[正面]
单词（大字）
词性

[背面（点击翻转）]
英文释义
例句（英文）
[译成中文] 按钮（按需显示中文释义和例句译文）
```

### 4.3 交互流程

1. 以卡片轮播形式展示 50 个词（可左右滑动）
2. 用户翻转每张卡片，查看释义
3. 全部 50 张看完后，解锁游戏入口
4. 进度条显示「已看 N/50」

## 五、游戏一：Clusters（Connections 式）

### 5.1 玩法概述

参考 NYT Connections。将 16 个词分成 4 组，每组 4 个词共享一个隐藏主题。

```
[16 个词格子，随机排列]
用户点击选择 4 个词 → 提交猜测
正确 → 该组词消除，显示主题名
错误 → 提示错误次数，可继续尝试
```

### 5.2 SAT 定制考点

| 分组颜色 | 典型主题类型 | 考点 |
|----------|-------------|------|
| 黄（最易） | 词性相同的同义词 | 同义词识别 |
| 绿 | 细微语义差异的近义词 | 近义词辨析 |
| 蓝 | 反义词组 / 对立概念 | 反义词 |
| 紫（最难） | 语境相关词 / 微妙联系 | 深度语义理解 |

### 5.3 关键机制

- **One Away 提示**：猜错时，若选中的词中有 3 个属于同一组，显示「差一步！」提示
- **错误上限**：最多允许 4 次错误，超出则显示答案（不影响进度记录）
- **颜色不透露难度**：分组完成前用户看不到颜色，只能看到题目词

### 5.4 内容要求

每日一套 Connections 题目，由 `ConnectionGroup` 模型存储：
- 每套 4 组，每组 4 词，共 16 词
- 主题（theme）、颜色（yellow/green/blue/purple）、难度（1-4）
- 词必须来自词汇池

## 六、游戏二：Five（Wordle 式）

### 6.1 玩法概述

参考 NYT Wordle。猜一个 5 字母 SAT 词，最多 6 次机会。

```
[6 行输入格，每行 5 格]
用户键入 5 字母 → 回车提交
每格变色：
  绿色 = 字母位置正确
  黄色 = 字母存在但位置错
  灰色 = 字母不在答案中
```

### 6.2 SAT 定制考点

- 答对需要同时具备**拼写能力**和**词义回忆能力**
- 猜词结束后（无论对错），显示目标词的完整词卡（含释义、例句）

### 6.3 字母着色算法（两遍扫描，处理重复字母）

```
第一遍：确认所有绿色（位置正确）
第二遍：对非绿色位置，按答案中剩余字母数量分配黄色
规则：键盘颜色只升不降（灰→黄→绿），已绿不覆盖
```

### 6.4 词库限制与处理

- Wordle 只能使用 5 字母词，会过滤掉大量 SAT 词
- 解决方案：从 SAT 词库中专项筛选 5 字母词，作为 Five 专用词库
- 每日一个答案词，从专用词库按难度轮换

### 6.5 提示机制

- 第 3 次猜错后，可选择显示「词义提示」（一句话描述）
- 第 5 次猜错后，可选择显示「首字母提示」

## 七、游戏三：Strands（Strands 式）

### 7.1 玩法概述

参考 NYT Strands。在字母网格中找出所有主题词，通过连续划选字母拼出单词。

```
[N×M 字母网格，字母填满整个格子]
用户划选相邻字母（上下左右斜均可）→ 形成单词
找到主题词 → 高亮标记
找到所有主题词 → 过关
```

### 7.2 SAT 定制考点

- 主题词按一个语义主题（如"表达坚持的词"）组织
- 找到词后，点击可查看释义
- 额外有一个「Spangram」词：贯穿整个网格，提示主题

### 7.3 网格生成约束

- 所有主题词在网格中路径不重叠
- 字母 100% 填满网格（无空格）
- 非主题词字母也可以组成普通词（干扰项）
- Phase 1 使用**预设盘面**（手工排布），Phase 2 再考虑算法生成

### 7.4 内容要求

每日一套 Strands 盘面，包含：
- 网格尺寸（建议 6×8 或 8×6）
- 完整字母矩阵
- 主题名称
- 各主题词及其网格路径坐标
- Spangram 词及路径

## 八、进度与激励

### 8.1 今日完成状态

首页显示三个游戏的完成徽章：
- 灰色 = 未解锁（需先学词）
- 黄色 = 已解锁但未完成
- 绿色 = 今日已完成

### 8.2 连续学习 Streak

- 记录连续完成词汇学习的天数
- 显示在首页顶部
- 不依赖游戏是否胜利（学词即算）

### 8.3 错词自动入生词本

- 游戏中猜错的词，自动加入 VocabNote（`addedFrom` 记录来源游戏）
- 生词本中的词会在后续游戏中优先出现（强化复习）

## 十、数据模型补充

`DailyWordSet`（每日 50 词集）：

```prisma
model DailyWordSet {
  id        String   @id @default(cuid())
  date      String   // YYYY-MM-DD
  wordIds   String[] // 50 个词 id
  isActive  Boolean  @default(true)
}
```

`ConnectionGroup`（Clusters 分组）：

```prisma
model ConnectionGroup {
  id          String   @id @default(cuid())
  theme       String   // 隐藏分类名
  color       String   // yellow/green/blue/purple
  wordIds     String[] // 4 个词
  difficulty  Int      // 1-4
  setId       String   // 属于哪个 daily set
}
```
